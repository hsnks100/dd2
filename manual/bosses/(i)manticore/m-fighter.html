MANTICORE VS FIGHTER

※ 이 곳의 방법이 공략법의 전부는 아닙니다. 참고는 하되 자신만의 방법을 개발 해 보길 바랍니다.

파이터의 맨티코어 최단 시간 클리어 기록 : 4:43
기본 공략법
아이템 칸을 맞출 시간이 부족하니 바로 단검을 던질 수 있도록 준비한다.
그리고 시작하자 마자 던지고 콤보를 넣어주면 된다. 쓰러졌다가 일어나면 단검 던지고 콤보를 넣어주기를 반복하면 쉽게 끝난다. 되도록이면 A공격은 3타까지만 넣도록 한다.


* 영상 정보 - 플레이어 crassus / 클리어 시간 4:31 *

이도류나 노멀소드나 방법은 차이가 없다.
해머를 이용한 공략법
해머 → 단검 순으로 아이템을 정리하고 시작하자 마자 해머부터 던져주면 2히트가 되면서 많은 데미지를 줄 수 있다. 이어서 단검을 던지고 콤보를 한두번 넣어주면 끝난다.

그런데 나중에 나올 디스플레이서 비스트에게 해머를 쓰는 공략법을 주로 쓰는데 그 전까지 다시 상점이 나오지 않기 때문에 여기서 해머를 쓰면 나중에 그만큼 느려진다. 비스트가 더 어렵기 때문에 그쪽에 해머를 쓰는게 좀더 많이 선택되는데 어느쪽에 쓸 것인지는 플레이어가 직접 판단하기 바란다.


* 영상 정보 - 플레이어 : 종언의 은하로 / 클리어 시간 4:43 *

꼭 백점프 콤보를 넣을 필요는 없다. 자신이 없으면 그냥 무난한 콤보를 넣자.
기본 무기만으로 공략하기
단검이 없다면 칼을 한번 휘둘러서 가드하면 위나 아래로 빠졌다가 돌진한 후의 뒤를 노리고 맞았다면 바로 콤보로 이어주면 된다. 쓰러졌다가 일어날 때 공격하면 가드하는 경우가 많으니 아래 영상처럼 점프 후에 방향전환 점프찍기를 해주자.

콤보는 '슬래쉬 → A → A → 슬래쉬 → 대공기'가 쓸만하다.


* 영상 정보 - 플레이어 crassus / 클리어 시간 4:25 *

영상에서 처럼 가죽이 떨어지면 자리를 바꿔야 한다.
보너스 영상


* 영상 정보 - 플레이어 : 작안의 아키하 / 클리어 시간 4:35 *

평범하게 잡는게 질렸다면 한번 하하 콤보를 넣어보는 것도 나쁘지 않다.




* 영상 정보 - 플레이어 : 달통닭이 / 클리어 시간 4:31 *

대거와 하하콤보를 이용한 공략이다. 시작에서 맨티코어가 가드하지 못하도록 대거를 1개 던지고
6하하 + 4대거를 반복으로 공략한다.(하하커맨드에 관한건 따로 설명하지 않는다)



* 영상 정보 - 플레이어 : 달통닭이 / 클리어 시간 4:33 *

위와 비슷하나 이도류를 사용했고, 첫턴에 대거 대신 대쉬하하로 가드를 무마시켰다.
이후 대쉬박 + (6하하 + 대거3) x3 +6하하 로 마무리 짓는다.



* 영상 정보 - 플레이어 : Crassus / 클리어 시간 4:31 *

대거만 이용해서 노다운-원킬 콤보를 만들었다. 헌데 공중에 뜨는걸 대거3타 끊어치기로 세워야하기때문에
어찌보면 하하보다 더 어렵다.(운작용때문에) 그냥 이리 될 수도 있다고 보기만 하자
