전반부


전반부는 이전 오버츄어와 색만 다르고 같은 구조다.




조금 전진하다 보면 셰도우 엘프가 뛰어온다.

 

등장하는 적 - 약 40초가 되면 전부 도망간다. 전부 물리치거나 도망가야 다음으로 진행 할 수 있다.

1인 왼쪽에서 셰도우 엘프 x2 → 한명만 남으면 왼쪽에서 셰도우 엘프 x2 + 오른쪽에서 셰도우 엘프 x2 → 두명만 남으면 왼쪽에서 창든 셰도우 엘프 x3 + 오른쪽에서 창든 셰도우 엘프 x3
2인 1인 플레이와 똑같다.
3인 왼쪽에서 셰도우 엘프 x2 → 한명만 남으면 왼쪽에서 셰도우 엘프 x3 + 오른쪽에서 셰도우 엘프 x2 → 두명만 남으면 왼쪽에서 창든 셰도우 엘프 x3 + 오른쪽에서 창든 셰도우 엘프 x3
4인 왼쪽에서 셰도우 엘프 x2 → 한명만 남으면 왼쪽에서 셰도우 엘프 x4 + 오른쪽에서 셰도우 엘프 x2 → 두명만 남으면 왼쪽에서 창든 셰도우 엘프 x3 + 오른쪽에서 창든 셰도우 엘프 x3

 

 



조금 전진하다 보면 오른쪽에서 헬 하운드가 튀어나온다. 스테이지 진행과는 상관없다.

 

등장하는 적 - 약 25초가 되면 전부 도망간다.

1인 오른쪽에서 헬 하운드 x2 → 한마리만 남으면 오른쪽에서 헬 하운드 x1
2인 1인 플레이와 똑같다.
3인 오른쪽에서 헬 하운드 x2 → 한마리만 남으면 오른쪽에서 헬 하운드 x2 →
한마리만 남으면 오른쪽에서 헬 하운드 x2
4인 3인 플레이와 똑같다.

 

 



한참 전진하다 보면 또 헬 하운드가 튀어나온다. 이번에는 전부 잡아야 진행이 된다.

 

등장하는 적 - 약 30초가 되면 전부 도망간다. 전부 물리치거나 도망가야 다음으로 진행 할 수 있다.

1인 왼쪽에서 헬 하운드 x2 + (잠시 뒤에) 오른쪽에서 헬 하운드 x1
2인 왼쪽에서 헬 하운드 x3 + (잠시 뒤에) 오른쪽에서 헬 하운드 x1
3인 왼쪽에서 헬 하운드 x3 + (잠시 뒤에) 오른쪽에서 헬 하운드 x2
4인 3인 플레이와 똑같다.

 

 











다시 한참 전진하다 보면 고블린, 헬 하운드, 셰도우 엘프가 종합선물세트로 등장한다.
이번 전반부의 마지막 적들이다. 마지막이라 그런지 무척 많이 나온다.

 

등장하는 적 - 약 50초가 되면 전부 도망간다. 전부 물리치거나 도망가야 다음으로 진행 할 수 있다.

1인 왼쪽에서 고블린 x4 → 한마리를 제거하면 오른쪽에서 돌팔매질 고블린 x3 → 두마리만 남으면 왼쪽에서 셰도우 엘프 x2 + 오른쪽에서 셰도우 엘프 x2 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 헬 하운드 x1 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 셰도우 엘프 x2 + 왼쪽에서 헬 하운드 x1 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 왼쪽에서 창든 셰도우 엘프 x2 (바로 칼을 뽑고 싸움에 참가) → 총 두마리만 남으면 오른쪽에서 창든 셰도우 엘프 x3 → 왼쪽에서 창든 셰도우 엘프 x3
2인 왼쪽에서 고블린 x4 → 한마리를 제거하면 오른쪽에서 돌팔매질 고블린 x3 → 두마리만 남으면 왼쪽에서 셰도우 엘프 x2 + 오른쪽에서 셰도우 엘프 x2 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 헬 하운드 x2 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 셰도우 엘프 x2 + 왼쪽에서 헬 하운드 x1 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 왼쪽에서 창든 셰도우 엘프 x2 (바로 칼을 뽑고 싸움에 참가) → 총 두마리만 남으면 오른쪽에서 창든 셰도우 엘프 x3 → 왼쪽에서 창든 셰도우 엘프 x3
3인 왼쪽에서 고블린 x4 → 한마리를 제거하면 오른쪽에서 돌팔매질 고블린 x3 → 두마리만 남으면 왼쪽에서 셰도우 엘프 x3 + 오른쪽에서 셰도우 엘프 x2 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 헬 하운드 x3 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 셰도우 엘프 x2 + 왼쪽에서 헬 하운드 x1 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 왼쪽에서 창든 셰도우 엘프 x2 (바로 칼을 뽑고 싸움에 참가) → 총 두마리만 남으면 오른쪽에서 창든 셰도우 엘프 x3 → 왼쪽에서 창든 셰도우 엘프 x3
4인 왼쪽에서 고블린 x4 → 한마리를 제거하면 오른쪽에서 돌팔매질 고블린 x3 → 두마리만 남으면 왼쪽에서 셰도우 엘프 x4 + 오른쪽에서 셰도우 엘프 x2 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 헬 하운드 x3 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 셰도우 엘프 x2 + 왼쪽에서 헬 하운드 x1 + 오른쪽에서 헬 하운드 x1 → 총 두마리만 남으면 왼쪽에서 창든 셰도우 엘프 x2 (바로 칼을 뽑고 싸움에 참가) → 총 두마리만 남으면 오른쪽에서 창든 셰도우 엘프 x3 → 왼쪽에서 창든 셰도우 엘프 x3

꺽어지는 부분에 문이 하나보이지만 불이 가로막고 있다.
불 내성의 반지가 있다면 닿더라도 피해는 입지 않지만 들어갈 수 는 없으며 아직까진 그냥 모양만 문일 뿐 일반 벽과 다름이 없다.

왼쪽에 보이는 위아래로 불이 있는 지역을 지나가려 하면 화살 트랩이 작동한다.
디스플레이서 클록이 없다면 돈 집는다고 무작정 달려들지 않도록 한다.

한번 작동한 위치는 다시는 작동하지 않으니 디스플레이서 클록을 가진 플레이어가 먼저 밟아서 작동시키고 그 위를 나머지 플레이어가 지나가거나 아니면 맨아래쪽에 붙으서 전진하거나 맨위를 점프해서 건너가도록 한다.

오른쪽에서 보이는 문은 마찬가지로 아직까진 들어갈 수 없다.
아래쪽에는 책 3종 세트와 향수가 놓여있다.

문이 하나 있고 좌우로 가고일이 늘어서 있다.
여기의 문은 바로 들어갈 수 있으며 가고일을 깨우지 않고 들어가려면 대쉬 점프로 들어가도록 한다. 시프라면 반 날다람쥐로 들어가도 된다.

가고일이 있는 지역에서 약간 오른쪽 지역을 시프 이외의 클래스가 접근하면 분쇄기가 작동한다. 또한 위아래로 불꽃이 있는 지역을 통과하면 화살 트랩이 작동하는데 둘다 맨아래로 붙어서 가는게 안전하다. 2인 이상 플레이시 시프를 제외한 클래스가 먼저 지나갔다면 분쇄기가 나왔다가 올라가는 것을 본 후에 지나가도록 한다.



가장 마지막 지역에 슈퍼 힐링 포션과 상자가 놓여있다. 상자에서도 힐링 포션이 두개 나온다.
