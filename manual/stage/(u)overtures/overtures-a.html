전반부


스테이지 9-C에서 텔'아린을 물리쳤기 때문에 다시는 나오지 않는다.




어느정도 가다보면 고블린들이 뛰어온다.

 

등장하는 적 - 약 37초가 되면 전부 도망간다. 전부 물리치거나 도망가야 다음으로 진행할 수 있다.

1인 왼쪽에서 고블린 x3 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 돌팔매질 고블린 x3 + 오른쪽에서 돌팔매질 고블린 x3
2인 1인 플레이와 똑같다.
3인 왼쪽에서 고블린 x4 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 돌팔매질 고블린 x3 + 오른쪽에서 돌팔매질 고블린 x3
4인 3인 플레이와 똑같다.

 

 

어느정도 가다보면 오른쪽에서 고블린이 한마리씩 튀어나온다. 스테이지 진행과는 상관이 없다. 한마리만 남으면 세마리가 더 등장하고 또 한마리만 남으면 세마리가 더 등장한다.
약 23초가 지나면 전부 도망간다.



조금 더 가다보면 이번엔 액스 놀이 등장한다.

 

등장하는 적 - 약 37초가 되면 전부 도망간다. 전부 물리치거나 도망가야 다음으로 진행할 수 있다.

1인 왼쪽에서 액스 놀 x2 + 오른쪽에서 액스 놀 x1 →
한마리를 제거하면 왼쪽에서 보우 놀 x3 + 오른쪽에서 보우 놀 x3
2인 왼쪽에서 액스 놀 x3 + 오른쪽에서 액스 놀 x1 →
한마리를 제거하면 왼쪽에서 보우 놀 x3 + 오른쪽에서 보우 놀 x3
3인 왼쪽에서 액스 놀 x3 + 오른쪽에서 액스 놀 x2 →
한마리를 제거하면 왼쪽에서 보우 놀 x3 + 오른쪽에서 보우 놀 x3
4인 3인 플레이와 똑같다.

 

 







한참 더 가다보면 고블린, 액스 놀이 차례로 뛰어온다.

 

등장하는 적 - 약 49초가 되면 전부 도망간다. 전부 물리치거나 도망가야 다음으로 진행할 수 있다.

1인 왼쪽에서 고블린 x3 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 고블린 x3 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 돌팔매질 고블린 x3 + 오른쪽에서 돌팔매질 고블린 x3 → 왼쪽에서 액스 놀 x2 + 오른쪽에서 액스 놀 x1 → 두마리만 남으면 왼쪽에서 액스 놀 x2 + 왼쪽에서 고블린 x4 → 오른쪽에서 보우 놀 x3 → 두마리만 남으면 왼쪽에서 보우 놀 x3 + 오른쪽에서 보우 놀 x3
2인 왼쪽에서 고블린 x3 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 고블린 x3 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 돌팔매질 고블린 x3 + 오른쪽에서 돌팔매질 고블린 x3 → 왼쪽에서 액스 놀 x3 + 오른쪽에서 액스 놀 x1 → 두마리만 남으면 왼쪽에서 액스 놀 x3 + 왼쪽에서 고블린 x4 → 오른쪽에서 보우 놀 x3 → 두마리만 남으면 왼쪽에서 보우 놀 x3 + 오른쪽에서 보우 놀 x3
3인 왼쪽에서 고블린 x4 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 고블린 x4 + 오른쪽에서 고블린 x1 → 두마리만 남으면 왼쪽에서 돌팔매질 고블린 x3 + 오른쪽에서 돌팔매질 고블린 x3 → 왼쪽에서 액스 놀 x3 + 오른쪽에서 액스 놀 x2 → 두마리만 남으면 왼쪽에서 액스 놀 x3 + 왼쪽에서 고블린 x4 → 오른쪽에서 보우 놀 x3 → 두마리만 남으면 왼쪽에서 보우 놀 x3 + 오른쪽에서 보우 놀 x3
4인 3인 플레이와 똑같다.

매직 유저는 궁극의 무기라 할 수 있는 위저드리 스태프를 얻을 수 있으니 필수적으로 들러야 할 방이다. 프로젝티드 이미지를 시전해서 쉽게 열 수 있다. 상자를 열려면 열쇠가 필요하다는 점을 잊지 않도록 한다.

다른 클래스는 불 내성의 반지가 없다면 라파엘의 동굴에서 소개했던 방법을 쓰도록 한다.
그러나 파이터, 드워프를 제외하면 여는 보람이 그다지 없을 것 이다.
