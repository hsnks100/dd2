전반부


스테이지 분위기는 절망의 숲과 약간 비슷하다.




약간 전진하면 셰도우 엘프가 나온다. 물리치지 않아도 계속 전진 할 수 있다.

 

등장하는 적 - 약 40초가 흐르면 전부 도망간다.

1인 오른쪽에서 셰도우 엘프 x3 → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 셰도우 엘프 x3 → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 셰도우 엘프 x3 → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 마법사 셰도우 엘프 → 오른쪽에서 창든 셰도우 엘프 x3 (이후 칼을 뽑고 싸움에 참가) → 한명만 남으면 오른쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 마법사 셰도우 엘프 → 왼쪽에서 창든 셰도우 엘프 x3 (이후 칼을 뽑고 싸움에 참가) → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 →전부 제거하면 왼쪽에서 마법사 셰도우 엘프 → 처음부터 다시 반복
2인 1인 플레이와 똑같다.
3인 오른쪽에서 셰도우 엘프 x3 → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 셰도우 엘프 x3 + 왼쪽에서 셰도우 엘프 x2 → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 셰도우 엘프 x3 → 한명만 남으면 왼쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 마법사 셰도우 엘프 → 오른쪽에서 창든 셰도우 엘프 x3 + 왼쪽에서 창든 셰도우 엘프 x2 (이후 칼을 뽑고 싸움에 참가) → 두명만 남으면 오른쪽에서 마법사 셰도우 엘프 → 전부 제거하면 오른쪽에서 마법사 셰도우 엘프 → 왼쪽에서 창든 셰도우 엘프 x3 + 왼쪽에서 창든 셰도우 엘프 x2 (이후 칼을 뽑고 싸움에 참가) → 두명만 남으면 왼쪽에서 마법사 셰도우 엘프 →전부 제거하면 왼쪽에서 마법사 셰도우 엘프 → 처음부터 다시 반복
4인 3인 플레이와 똑같다.

 

 

여기까지 오면 화면이 더이상 스크롤 되지 않으며 자이언트 비틀이 먼저 나온 다음 셰도우 엘프가 나온다.셰도우 엘프를 전부 물리쳐야 (혹은 도망쳐야) 다음으로 진행할 수 있다.

오른쪽에 슈퍼 힐링 포션이 놓여있고 3인 플레이 이상이라면 힐링 포션이 하나 더 놓여있다.

만약 초반에 나오는 셰도우 엘프중 창든 셰도우 엘프까지 봤다면 여기서 바로 다음으로 진행 시킬 수 있다. 그러나 보스(그린 드래곤)는 보스대로 나오고 이곳의 자이언트 비틀과 셰도우 엘프가 그대로 나오기 때문에 좋은 방법은 아니다.

 

등장하는 적 - 약 45초가 흐르면 도망간다.

1인 오른쪽에서 자이언트 비틀 x10 (처음엔 한마리씩 띄엄띄엄 나오다가 점점 빨라진다.) →
오른쪽에서 셰도우 엘프 x1 + 왼쪽에서 셰도우 엘프 x1 → 한명만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 자이언트 비틀 x10 → 오른쪽에서 셰도우 엘프 x1 + 왼쪽에서 셰도우 엘프 x1 → 한명만 남으면 오른쪽에서 궁수 셰도우 엘프 x3 → 오른쪽에서 자이언트 비틀 (아무리 빨리 잡아도 시간 한계 때문에 더 이상 확인 불가)
2인 1인 플레이와 똑같다.
3인 오른쪽에서 자이언트 비틀 x10 (처음엔 한마리씩 띄엄띄엄 나오다가 점점 빨라진다.) →
오른쪽에서 셰도우 엘프 x1 + 왼쪽에서 셰도우 엘프 x1 → 한명만 남으면 왼쪽에서 궁수 셰도우 엘프 x2 + 오른쪽에서 궁수 셰도우 엘프 x3 → 왼쪽에서 자이언트 비틀 x10 → 오른쪽에서 셰도우 엘프 x1 + 왼쪽에서 셰도우 엘프 x1 → 한명만 남으면 왼쪽에서 궁수 셰도우 엘프 x2 + 오른쪽에서 궁수 셰도우 엘프 x3 → 오른쪽에서 자이언트 비틀 (아무리 빨리 잡아도 시간 한계 때문에 더 이상 확인 불가)
4인 3인 플레이와 똑같다.

최초로 나오는 드래곤 이다.

시작하자마자 브레쓰를 내뿜는데 그린 드래곤 바로 아래에서 그림자보다 약간 위쪽에 위치하도록 한다. 그림자와 겹치면 브레쓰에 맞게 된다.

망치를 남겨왔다면 쉬운 싸움이 된다. 바로 망치를 다 던지고 A 1~2히트 후 강공격을 반복 해 3주면 브레쓰만 내뿜다 끝날 것 이다. 망치가 없을 때가 좀 문제인데 브레쓰가 끝날 때 까지 열심히 공격하다가 브레쓰를 다시 내뿜을 때 까지 매직 미사일이나 아이스 스톰을 쓰도록 한다.

만약 주문이 하나도 안남아 있다면 멀리 도망가서 화살을 점프해서 쏴서 견제하도록 한다.
